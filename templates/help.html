Help & Information

About the Dashboard

This application is a lightweight, web-based dashboard for monitoring an HCL Workload Automation (HWA) or Tivoli Workload Scheduler (TWS) environment via its REST API.

It runs as a background process with a system tray icon, and the dashboard is accessed through a local web URL.

Configuration

To connect to your HWA/TWS instance, you must provide your connection details.

  1. Click the Cog () icon in the top right of the dashboard.
  2. Fill in your HWA/TWS Hostname, Port, Username, and Password.
  3. The Verify SSL checkbox controls whether the application should validate the SSL certificate of the HWA server. Uncheck this for environments with self-signed certificates.
  4. Click Save Configuration. The settings are saved internally and will be used for all future connections.

Features

  • Real-time Monitoring: The dashboard automatically refreshes to show the latest job stream statuses.
  • System Tray Icon: The application runs in the background. You can open the dashboard or exit the application from the tray icon menu.
  • Start with Windows: (If on Windows) You can set the application to start automatically when you log in via the tray icon menu.
