HWA Neuromorphic Dashboard Help

This dashboard provides a real-time overview of your HCL Workload Automation (HWA) or Tivoli Workload Scheduler (TWS) environment.

Features

  • Dynamic Widgets: The main dashboard displays key metrics. You can click on these widgets to see more details.
  • Job and Workstation Grids: View all your job streams and workstations at a glance.
  • Configuration: Use the configuration page to set up the connection to your HWA/TWS environment.
  • Movable Windows: All pop-up windows can be dragged around the screen.

Troubleshooting

If you are having trouble connecting, please check the following:

  • The hostname and port are correct.
  • The username and password are correct.
  • If your HWA/TWS environment uses a self-signed SSL certificate, make sure to uncheck the "Verify SSL Certificate" box in the configuration.
